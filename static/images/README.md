## Image Optimization Tips

1. **Compression**: Use online tools like TinyPNG or ImageOptim to reduce file size
2. **Format**: Use WebP for better compression, with JPG fallback
3. **Responsive**: Consider creating multiple sizes for different devices
4. **Alt Text**: Always include descriptive alt text in the HTML

## File Naming Convention

Use lowercase, hyphenated names:
- ✅ hero-left.jpg
- ✅ team-member-1.jpg
- ❌ HeroLeft.jpg
- ❌ Team_Member_1.jpg
