# Logos Directory

This directory contains logo assets and brand identity files for Grissom Corporate Services.

## Logo Files

### grissom-logo.svg
- **Purpose**: Main company logo
- **Format**: SVG (Scalable Vector Graphics)
- **Usage**: 
  - Navigation bar
  - Footer
  - Brand materials
  - Favicon (convert to ICO)

### grissom-logo-white.svg
- **Purpose**: Logo for dark backgrounds
- **Format**: SVG
- **Usage**:
  - Dark themed sections
  - Print on dark backgrounds

### grissom-logo-minimal.svg
- **Purpose**: Simplified mark version
- **Format**: SVG
- **Usage**:
  - Favicon
  - Small icon displays
  - Social media avatars

## Logo Guidelines

### Do's ✅
- Use the provided SVG format for web (infinitely scalable)
- Maintain minimum size of 24px for display
- Use appropriate contrast ratios for accessibility
- Keep white space around the logo

### Don'ts ❌
- Don't distort or skew the logo
- Don't change brand colors without approval
- Don't use raster versions when vector is available
- Don't place on busy backgrounds without proper contrast

## File Naming Convention

Use lowercase, hyphenated names:
- ✅ grissom-logo.svg
- ✅ partner-logo-1.svg
- ❌ GrisomLogo.svg
- ❌ Partner_Logo_1.svg

## SVG Best Practices

1. **Inline SVGs**: Use for navigation/header logos
2. **SVG as Image**: Use `<img>` tag for consistency
3. **SVG as Background**: Use in CSS for scalability
4. **Optimization**: Remove unnecessary code and metadata

## Creating SVG Logos

Tools:
- Adobe Illustrator
- Sketch
- Figma
- Inkscape (free)
- Adobe XD

Always export as "Optimized SVG" to reduce file size.
